# ExamPulse Frontend

React + Vite frontend for the ExamPulse AI-powered exam preparation platform.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

The app will be available at `http://localhost:5173`

## Build

```bash
npm run build
```

## Pages

- **Home** (`/`) - Landing page
- **Upload** (`/upload`) - Upload exam papers
- **Analysis** (`/analysis`) - View analysis results
- **Expected Paper** (`/expected-paper`) - Generate expected paper
- **Study Logs** (`/study-logs`) - Create study log entries
- **Smart Plan** (`/smart-plan`) - View AI-powered study plan

## Components

- `Navbar` - Navigation bar
- `FileUpload` - File upload component
- `TopicChart` - Topic frequency chart (Recharts)
- `PlanCard` - Smart plan display card

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Recharts
- Axios
- React Router
